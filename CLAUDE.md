## Project Overview

家計簿アプリ（Kakeibo App）— a Japanese household budget tracking application.

## Git Workflow

**After every code change, commit and push to GitHub.**

```bash
git add <changed files>
git commit -m "concise description of change"
git push origin main
```

- Commit messages should be in English, concise, and describe *what* changed
- Push immediately after each commit — do not batch multiple sessions of work before pushing
- Never force-push to `main`
